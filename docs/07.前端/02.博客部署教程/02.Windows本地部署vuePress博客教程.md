1、下载并安装`Nodejs 20.8.1`

```shell
nvm install 20.8.1
nvm use 20.8.1
```

2、全局安装yarn

```shell
npm install -g yarn
```

3、下载vuePress包

```shell
yarn add -D vuepress
```

4、启动博客

```shell
yarn dev:win
```
