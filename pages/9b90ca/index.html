Nodejs的环境安装

# 1、安装Nodejs

# Windows本地安装

可以在Nodejs的官网下载exe安装文件进行安装，下载地址是：Nodejs官网下载地址 (opens new window)，也可以使用nvm进行安装，nvm是Nodejs的管理工具，通过它可以下载各种版本的Nodejs并迅速进行切换。

使用nvm安装、管理Nodejs的步骤如下：

1、下载并安装nvm，下载地址是：nvm下载地址 (opens new window)。

2、为nvm配置国内镜像源，只需要修改settings.txt文件即可：

root: D:\Software\java\nvm
path: D:\Software\java\nodejs
npm_mirror https://npmmirror.com/mirrors/npm/
node_mirror https://npmmirror.com/mirrors/node/
1
2
3
4

3、安装好nvm后，查看版本：

nvm -v
1

4、罗列出nvm中可安装的Nodejs版本：

nvm list available
1

5、选择一个版本的Nodejs进行安装：

nvm install 8.9.4
1

6、安装完Nodejs后，需要使用才能生效：

nvm use 8.9.4
1

7、查看已安装的所有Nodejs版本：

nvm list
1

8、卸载Nodejs：

nvm uninstall 8.9.4
1

# Windows中不同版本Nodejs的切换问题

windows中不同版本的Nodejs在切换时，可能会因为包冲突产生各种莫名其妙的问题，因此在切换前，最好先删除以下文件：

C:\Users\12136\.npmrc
C:\Users\12136\.yarnrc
C:\Users\12136\AppData\Roaming\npm
C:\Users\12136\AppData\Roaming\npm-cache

# 2、使用npm

npm是随同Nodejs一起安装的包管理工具，能解决Nodejs代码部署上的很多问题，常见的使用场景有以下几种：

  • 允许用户从npm服务器下载别人编写的第三方包到本地使用。
  • 允许用户从npm服务器下载并安装别人编写的命令行程序到本地使用。
  • 允许用户将自己编写的包或命令行程序上传到npm服务器供别人使用。

使用npm下载包、启动项目的步骤如下：

1、查看npm版本：

npm -v
1

2、查看当前镜像源：

npm config get registry
1

3、设置国内镜像源：

npm config set registry https://registry.npmmirror.com
1

4、下载包：

npm install # 下载所有包
npm install xxx # 下载指定包
1
2

5、执行项目的部署脚本，其中dev指的就是脚本，可以是编译脚本，也可以是部署脚本：

npm run dev
1

# 3、使用yarn

Yarn是由Facebook、Google、Exponent 和 Tilde 联合推出的一个新的 JS 包管理工具 ，它是为了弥补 npm 的一些缺陷而出现的。

使用yarn下载包、启动项目的步骤如下：

1、通过npm安装yarn

npm install -g yarn
1

2、查看yarn版本

yarn -v
1

3、查看当前镜像源

yarn config get registry
1

4、设置国内镜像源

yarn config set registry https://registry.npmmirror.com
1

5、下载包

yarn install # 下载所有包
yarn install xxx # 下载指定包
1
2

6、执行项目的部署脚本

yarn dev:win
1
